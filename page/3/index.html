发表于 2021-06-10

压缩感知

压缩感知和数据压缩相反，数据压缩是已知高维的数据，将其通过低维的表示出来，以达到压缩数据量的目的。而压缩感知则是在数据获取的时候，通过使用Unsample的数据，即采样数没有真实数据维数多，来还原真实数据的，使得还原的数据和真实数据较好符合的一门技术。换句话说，压缩感知，是通过压缩过的感知手段来还原真实数据。

Nyquist–Shannon(奈奎斯特-香农)采样定理

定理内容 :如果一个系统以超过信号最高频率至少两倍的速率对模拟信号进行均匀采样，那么原始模拟信号就能从采样产生的离散值中完全恢复。

epoll-linux

发表于 2021-03-27 更新于 2021-05-27

参考资料(http://www.52im.net/thread-1939-1-1.html)

epoll主要是用来解决服务器单线程处理用户请求时若请求事务过重造成单位时间处理事务少，给其他用户造成长时间等待的情形。

通过epoll可以采用Reactor模式，将接受请求和业务处理分割开来，实现更高的并发数量和处理效率。
